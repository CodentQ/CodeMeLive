    Welcome to Code Me Live.

    Code Me Live is a open source project which helps website owners or article writers to display HTML+CSS+JS code live & editable on their own website. If user interested to use API then we have an under construction website, It will provides API to display live and editable code after save that code on API website.

    This project is developed on pure HTML, CSS, jQuery because user can use it on any platform. By help of this open source project user can develop own dynamic player/editor for own website and with own design.
